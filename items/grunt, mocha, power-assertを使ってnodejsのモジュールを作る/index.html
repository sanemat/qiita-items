simple-mochaがよさそうだけど、mochaのrequireオプションをpower-assert関連で使いたいので、requireオプション使えるgrunt-mocha-testを使う。

npm install grunt-mocha-test --save-dev

power-assert

npm install power-assert intelli-espower-loader --save-dev

assertをpower-assertに変える。
grunt-mocha-test のrequire オプションにintelli-espower-loader渡す。

package.jsonに "directories": { "test": "test/" },付け足して終わり。
